DEVS-TOSSIM: A DEVS framework for simulation of TinyOS wireless sensor networks

DEVS-TOSSIM is a simulation framework, which implements the DEVS formalism, to test and simulate wireless sensor networks that use the TinyOS operating system.

Features

  • Heterogeneous wireless sensor networks: DEVS-TOSSIM can simulate heterogeneous wireless sensor networks, which implies that in the same simulation can coexist nodes that run different software and hardware.
  • Duty-cycling: the transceiver model supports duty cycling.
  • User interface: simple user interface to control the simulation; the interface allows the user to control time during the simulation, for example: execute the simulation in real-time, as fast as possible or step-by-step.
  • Integration with external systems: functionality is provided to integrate the simulator with external hardware or software systems.
  • Extensible framework: the framework allows the replacement of the sensor component of a node, the addition of mobility, the implementation of a new radio model or modeling a complex sensing environment.
